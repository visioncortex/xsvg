<xsvg-view> — vanilla embed

The barebone viewer. In production you include one self-contained script and drop the element into any page — it renders like an image, no other code required: 

<script src="xsvg.js"></script>
<xsvg-view src="diagram.xsvg"></xsvg-view>

This dev page registers the element from source instead. Inline documents live in a <script type="application/xsvg+xml"> data island so the HTML parser leaves the custom XML alone.
